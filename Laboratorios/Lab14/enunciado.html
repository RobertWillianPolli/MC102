Caça-Números

Caça-palavras é um jogo de passatempo que consiste em encontrar palavras em uma grade retangular, onde as letras estão dispostas de forma aparentemente aleatória. A medida que o tamanho da grade cresce, encontrar novas palavras fica cada vez mais difícil. Variações desse jogo podem ser criadas alterando os valores presentes na grade ou o padrão que deve ser buscado.

Uma empresa renomada no mercado de jogos de passatempo está desenvolvendo a ideia de uma nova variação do jogo chamada de Caça-Números. Nessa nova variação, cada posição da grade possui um número inteiro, sendo os valores possíveis de 1 (um) até 9 (nove). Além disso, ao invés de ser fornecida uma palavra para ser buscada na grade, um padrão formado por uma sequência de caracteres (string) é dado onde cada caractere representa uma característica de um número que deve ser encontrado, sendo eles:

  • A (any): qualquer número.
  • E (even): números pares.
  • O (odd): números ímpares.
  • C (composite): números compostos (4, 6, 8, 9).
  • P (prime): números primos (2, 3, 5, 7).
  • S (singular): número 1.
  • T (three): números múltiplos de três (3, 6, 9).

A busca na grade pode ser feita na horizontal (para cima ou para baixo), vertical (para a direita ou para a esquerda) e nas quatro diagonais. Além disso, os números poderão ser reutilizados para formar o padrão buscado. Por exemplo, considerando a grade de números a seguir e buscando pelo padrão ETETET, os números 2 e 3 poderão ser reutilizados para formar o padrão buscado.

1 1 1
1 2 3
1 1 1

O objetivo dessa versão será determinar as posições iniciais (linha e coluna), a partir das quais é possível formar o padrão desejado. A grade de números e o padrão desejado são produzidos de forma automática pela empresa. Entretanto, nenhuma verificação é realizada para saber se é possível ou não encontrar o padrão informado. A empresa não quer disponibilizar um jogo para os clientes em que não é possível encontrar uma solução, pois isso seria horrível para a reputação dela no mercado. O diretor da empresa, sabendo desse problema, solicitou que um(a) programador(a) desenvolva um programa para verificar se é possível encontrar o padrão desejado ou não na grade. Sabendo das suas habilidades com programação, você foi encarregado(a) de realizar essa tarefa.

A entrada do programa consiste em: L linhas representando a grade, sendo que cada linha possui C números separados por um único espaço. Por fim, é dada uma string indicando o padrão que deve ser buscado.

Como saída, seu programa deverá exibir as posições na grade (linha e coluna) em que é possível formar o padrão a partir desse ponto inicial. As posições deverão ser exibidas seguindo a ordem raster (de cima para baixo, da esquerda para direita). Note que a posição no canto superior esquerdo da grade é a posição (1,1). Caso não seja possível identificar o padrão na grade, seu programa deverá imprimir a seguinte mensagem:

Nenhum padrao encontrado!
